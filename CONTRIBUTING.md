# Contributing to PayWen

We welcome contributions from the community.  
Please follow these guidelines to ensure a consistent and secure workflow.

## 1. Fork and Clone

```bash
git clone https://github.com/yourusername/paywen.git
cd paywen
```

## 2. Branching

Create a descriptive feature branch.

```bash
git checkout -b feature/add-withdraw-endpoint
```

## 3. Code Style

- Use TypeScript where possible
- Follow clean function naming
- Keep commits small and descriptive

## 4. Submitting a Pull Request

When ready, push your branch and open a Pull Request:

```bash
git push origin feature/add-withdraw-endpoint
```

Describe what you changed and why.  
All contributions are reviewed for correctness and security.

---

For security reports, please contact  
`security@paywen.app`
